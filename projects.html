Research Experience

Unfortunately, I did not get the desired result, but I could learn a lot from it. In addition, I have experience working on various AI projects while participating in competitions.

Paper

    A study on conditional music generation with GAN - PDF

    This paper is about a model that adds genre conditions to the "Musegan" model to generate songs based on the condition.

    This paper was written based on my first research experience as an undergraduate research student. I conducted this research through a university program called the 'Challenge Semester,' where I had to produce results within a given deadline. However, after implementing the model, the results were poor, and since I couldn't change the research topic, I eventually felt like I was doing it unwillingly. This made it a very disappointing study.

Projects

    Project 1 - Palletizing Robot

    This project involves creating a palletizing robot that recognizes and moves objects. Using a model based on 'YOLACT,' the robot calculates coordinate values to recognize objects and then controls the motor via Arduino to move them to designated positions.

    Project Duration

    2022.03~2022.08

    Work I Did

    • Implemented an object-grabbing algorithm
    • Integrated the camera and controlled motors using Arduino

    Lessons Learned / Improvements

    • Since the rotation angle of the robotic arm varies depending on the object's location, segmentation is more efficient than object detection
    • By improving the grabbing algorithm, the robot can be enhanced to handle objects of various shapes
    • By calculating joint angles using tools such as a linkage simulator and controlling the robotic arm accordingly, the palletizing robot can be used even in narrow spaces

    Project 2 - Self-driving Electric scooter System

    This is a self-driving project for electric scooters. Using the 'YOLO v5 model, the system detects collision risk factors such as pedestrians or cars through a camera. Additionally, it identifies dangerous areas, such as stairs or deep cuts, using LiDAR. If a risk is detected, the motor is controlled via Arduino to stop the kickboard. We have implemented a system to ensure the safety of electric scooter passengers.
