Projects

  • Project 1 - Palletizing Robot

    • This project involves creating a palletizing robot that recognizes and moves objects. Using a model based on 'YOLACT,' the robot calculates coordinate values to recognize objects and then controls the motor via Arduino to move them to designated positions.

  • Project 2 - Self-driving Electric scooter System

    • This is a self-driving project for electric scooters. Using the 'YOLO v5 model, the system detects collision risk factors such as pedestrians or cars through a camera. Additionally, it identifies dangerous areas, such as stairs or deep cuts, using LiDAR. If a risk is detected, the motor is controlled via Arduino to stop the kickboard. We have implemented a system to ensure the safety of electric scooter passengers.
